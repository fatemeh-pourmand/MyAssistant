# MyAssistant
A personal assistant for managing tasks, sending scheduled messages, special calls, and reminders.
Keep your tasks and notes organized and never miss important events or deadlines.

---

## Features

- **Scheduled Calls:** Make a call to a specific person at a set date and time.
- **Scheduled Messages:** Send custom messages to specific people automatically.
- **Task Reminders:** Receive alerts for important tasks or events.
- **Note Organization:** Automatically sort notes and tasks based on date and time.
- **Automatic Notifications:** Get notified when a task or note is due.
